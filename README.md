# K-means Automation

## Usage
```
python3 kmeans.py [-r | --range] [-s | --start] [-e | --end] [-n | --number] [-h | --help]
```

## Optional
```
-r|--range:	K range to check.			Example: -r 4,6		Default: 3,7
-s|--start:	Start point to generate points from.	Example: -s 2,3		Default: 0,0
-e|--end:	End point to generate points to.	Example: -e 100,100	Default: 1000,400
-n|--number:	Number of points to generate.		Example: -n 150		Default: 100
-h|--help:	To display extended help
```
